# Hero Background Video

## วิธีเพิ่มวิดีโอพื้นหลัง

วางไฟล์วิดีโอชื่อ `hero-background.mp4` ในโฟลเดอร์นี้

### ข้อกำหนด:
- ชื่อไฟล์: `hero-background.mp4`
- แนะนำ: ความละเอียด 1920x1080 หรือสูงกว่า
- ขนาดไฟล์: ไม่เกิน 10MB เพื่อความเร็วในการโหลด
- ความยาว: 10-30 วินาที (จะเล่นซ้ำอัตโนมัติ)

### หมายเหตุ:
- หากไม่มีวิดีโอ Hero Section จะแสดงรูปภาพ fallback จาก Unsplash แทน
- วิดีโอจะเล่นอัตโนมัติ (autoplay) และซ้ำ (loop) โดยไม่มีเสียง (muted)
